# RDR

A website to monitor the operational status and capacity of an Urgent & Emergency Network.

## Features

- Real-time status monitoring
- Capacity tracking
- Network health dashboards
- Alert management

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running the Project

```bash
npm start
```
